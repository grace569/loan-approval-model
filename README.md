# Lending Club Loan Approval Optimization

Objective:
Build a cost-sensitive ML model to optimize approval decisions under asymmetric risk.

### 1. Problem Statement

> A marketplace lending platform must decide whether to approve or reject loan applications. Incorrect approvals result in credit losses, while overly conservative rejection reduces revenue. This project builds a cost-sensitive machine learning model to optimize loan approval thresholds under asymmetric risk.
> 

### 2. Define the Business Objective

- Minimize expected loss
